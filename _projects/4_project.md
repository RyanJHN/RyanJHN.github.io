**Role:** Independent researcher  
**Period:** May 2024 - Aug 2024

Developed a prototype recommendation system that analyzes employer reviews to generate personalized company recommendations.

- Collected employer review data from Handshake using automated web extraction.
- Applied sentiment analysis and text feature extraction to model user preferences from review content.
- Implemented similarity-based recommendation using binary feature matrices and cosine similarity to rank relevant employers.
