**Role:** Research Assistant  
**Supervisor:** Professor Pengyu Hong  
**Period:** Apr 2025 - Present

Doc2Agent converts unstructured REST API documentation into validated Python tools so LLM agents can call real-world APIs more reliably.

- Ran a full evaluation pipeline over 744 real-world API endpoints, all WebArena environments, and scientific APIs.
- Achieved a **55.1% improvement** over baseline API-calling agents.
- Integrated Python execution into a CodeAct-style agent for conditional, multi-step tool use.
- Fine-tuned Gemma-3 models for structured API information extraction.
