MindGames is a NeurIPS-hosted challenge focused on social reasoning, theory of mind, hidden-information reasoning, and strategic planning in interactive multi-agent games.

- Built a custom LLM-driven agent with personality traits encoded into model state.
- Implemented trait-steering behavior logic for consistent decision-making styles.
- Managed simulation setup and execution for iterative agent evaluation.
